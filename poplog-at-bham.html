Getting help regarding Poplog, Pop-11, Ved/XVed

Please note that none of the computer officers is an expert on Poplog and they will probably not be able to respond to requests for help. So do not try sending questions about it to the local support email list.
There is a local email list that can be used to send requests for help regarding poplog. This is the poplog-users email list. Anyone can join this list.
There is a more general list which is also used by people outside this university pop-forum. This is linked to the comp.lang.pop network news group (world wide bulletin board). You can send requests to either of those.
